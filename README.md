### Developed and Maintained by suffix
#

### Library Overview (Functions)
```
Console:
    Initialize
    GetConsoleSize
    SetConsoleSize

    Print:
        RGB
        GradientRGB

    Text:
        SetTextColor
        ResetTextColor
        CenterText

    Cursor:
        SetVisible
        SetSize
```

#

### Building

```
(In src directory)
mkdir build
cd build
cmake ..
cmake --build . --config Release
```
